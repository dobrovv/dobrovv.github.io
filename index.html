My Projects

Portable Wireless LED-Strip

JUN 2021 - JUL 2021 Android, Arduino GitHub github.com/dobrovv/LED_Controller
I designed and developed a light and portable controller for LED strips that can be firmly attached to a surface or carried around and controlled remotely from an Android app. The app allows customization of patterns, colors and brightness using a custom-made BLE driver for the Arduino Beetle microcontroller.

Tablet Kiosk

SEP 2020 - MAY 2021 AWS Aurora, WebComponents, Node.js FitPlans.ca
As part of the team at FitPlans, I created a web app for the kiosk dine-in tablets for the Montreal startup using the cutting-edge Web Components and HTML5. The app being responsive could function on multiple screen resolutions and run on the in-store dine-in tablets as well as mobile devices. The web app featured a gallery of fitness meals that could be ordered in the e-store, along with a caloric calculator that would recommend the caloric needs to achieve fitness goals based on a short interactive questionnaire.

HoloGama - CRM

JUN 2019 - OCT 2019 React, Express.js, Node.js, MySQL HoloGama.com
I have been working with a holographic advertisement start-up, HoloGama, to extend their CRM system (Customer Relationship Management System). The main objective was to enable the upload of advertisement media files to be displayed by the holographic projectors via the web UI . These projectors are located in several boutiques in Montreal. As part of the project, I implemented the API from the design specifications, wrote the documentation for future development, extended the endpoints and added validation, and implemented the web UI.

ATLAS - Tracker Wearable

SEP 2018 - APR 2019 Android, Arduino, SQLite, FlaskJS, GPRS, GSM, Google API GitHub github.com/dobrovv/Atlas
Our team of five has developed a wearable GSM tracker that can be used to locate missing items, pets, or children. With dimensions of 5 x 5 x 3 cm and a battery life lasting up to several days, our Arduino-based board could be firmly installed in a custom-built 3D printed case, and withstand even harsh weather conditions. My contribution to the project was developing our cloud API and establishing the connection between the GPRS tracker and the Android app, which allowed the trackers to be registered online, and tracked conveniently via our Android app and the Google Maps extension.
